## Setup & Build

### Clone the repository
```bash
git clone https://github.com/BaptisteBoulan/RayTracing.git
cd minimalRenderze
```

### First-time compilation
```bash
mkdir build
cmake -B build
cmake --build build --config Release
./build/Release/renderze.exe
```

### Rebuild & run
```bash
cmake --build build --config Release
./build/Release/renderze.exe
```

### Just run
```bash
./build/Release/renderze.exe
```
